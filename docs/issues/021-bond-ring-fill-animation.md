## AI Triage Brief

**Category**: enhancement. Adds the bond ring fill animation per `DESIGN.md` — a 600ms ease-out-quart sweep when bond score changes after an AI Update.

**State**: ready-for-agent. Spec pinned in `DESIGN.md → Components → Bond ring → Animation` and `DESIGN.md → Motion`.

**Codebase notes**:
- `BondRing` introduced in issue 012 renders the arc statically.
- Bond score updates happen in `AppController.commitAiUpdate(...)` (from issue 018) which sets the new `bondScore` on the affected `Connection`.
- `MediaQuery.of(context).disableAnimations` reads OS-level reduce motion.

**Implementation notes for the agent**:
- Convert `BondRing` to use an `AnimatedBuilder` with an `AnimationController` driven by score changes. Track the previously-rendered score in `_BondRingState`. When `widget.connection.bondScore` differs from the previous value, animate from old to new over 600ms with `Curves.easeOutQuart`.
- The arc's `value` property comes from the animated `Tween<double>(begin: oldScore/100, end: newScore/100)`.
- Optional saturation pulse: while animating, render the arc at 80% opacity, snapping to full opacity at rest. Implement only if it doesn't compromise readability.
- **Reduced motion**: when `MediaQuery.of(context).disableAnimations == true`, skip the controller — render the new score immediately. Confirm via test.
- Trigger surface: anywhere `BondRing` is mounted with a different `bondScore` than before. Riverpod's `ref.watch` on the connection rebuilds the widget; the animation runs when `didUpdateWidget` detects the change.
- DO NOT animate the arc on first mount (only on subsequent score changes). First mount renders the static target arc immediately, no sweep. Otherwise every list-row scroll triggers fake updates.
- Tests:
  - Pump the People tab. Update a contact's bond score via controller. Verify the arc value moves toward the new value (use `tester.pump(Duration(milliseconds: 100))` checks).
  - With `MediaQuery(data: MediaQueryData(disableAnimations: true), ...)`, the arc snaps immediately.
  - First-mount test: pump fresh widget; arc renders at the target value with no animation.
